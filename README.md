Perform code fix and review

## Install

Via Composer

``` bash
composer require funivan/cs:dev-master
```

## Usage

```sh
  # run review tool
  ./vendor/bin/cs.php review;

  # run fixer tool
  ./vendor/bin/cs.php fix;

```

## Testing

``` bash
    ./vendor/bin/phpunit
```
