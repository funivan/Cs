# Line ending

## Fixer
  Replace invalid line endings

  Name: `line_ending_fixer`

  *Before*:
  ```
...echo 1;\r\n
...\r\n
...echo 2;\r\n
  ```
  *After*:
  ```
...echo 1;\n
\n
...echo 2;\n
  ```

## Review
  Find lines with invalid line ending symbol

  Name: `line_ending_review`
