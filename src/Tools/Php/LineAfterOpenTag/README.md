# Line After Open Tag

## Fixer
  Set 1 empty line after php open tag inside php files

  Name: `php_line_after_open_tag_fixer`

  *Before*:
  ```<?php
        echo 1;
  ```

  *After*:
  ```<?php

        echo 1;
  ```


## Review
  Check for 1 empty line after php open tag


  Name: `php_line_after_open_tag_review`
